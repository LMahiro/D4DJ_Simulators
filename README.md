A semi-finished D4DJ simulator.<br />
Supports features such as creating charts, playing music, watching animations, and playing charts.<br />

模拟器完成了谱面播放器的基本代码<br />
*若想要播放谱面，请尝试如下步骤∶<br />
(在视频中也有介绍)<br />
1. 打开菜单
2. 点击关于软件
3. 点击简介分类下的日高沙织Q版人物10次
4. 开启调试模式
5. 重启模拟器后点击浮窗获取路径
6. 重命名谱面文件为chart.txt，音频文件为music.wav
7. 将文件放入路径中
8. 打开菜单
9. 点击测试用途按钮后将会跳转场景
   <br />
   *默认为Auto模式
